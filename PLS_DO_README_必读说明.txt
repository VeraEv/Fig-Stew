Fig Stew Desktop Pet / 蜜果羹桌宠
====================================

Please see below for English README:


这是一个蜜果羹桌宠。蜜果羹什么都不会做，除了生活在你的电脑桌面上。


1. 应该下载哪个版本？

- Apple Silicon Mac（M1 / M2 / M3 / M4，macOS Catalina 10.15 或更新）：使用
  Fig Stew-0.1.0-arm64-adhoc.dmg

- Intel Mac（macOS Catalina 10.15 或更新）：使用
  Fig Stew-0.1.0-x64-adhoc.dmg

- Windows 64 位电脑：使用
  Fig Stew Setup 0.1.0-win64.exe

如果你不确定自己的 Mac 是哪一种：
点击屏幕左上角 Apple 图标 -> About This Mac / 关于本机。
如果处理器写着 Apple M1/M2/M3/M4，就是 Apple Silicon。
如果处理器写着 Intel，就是 Intel Mac。

如果你的电脑不在以上列表中，请邮件联系：figstewissoyummy@proton.me，我会发给你适合的版本。


2. Mac 如何安装和打开？

注：这个应用目前没有 Apple Developer ID 正式签名，所以 macOS 可能会出现安全提示，也会让安装过程更加复杂。这是正常的。

1. 打开对应的 .dmg 文件。
2. 把 Fig Stew 拖到 Applications / 应用程序 文件夹。
3. 第一次打开时，请不要直接双击，请在 Finder / 访达 中打开 Applications / 应用程序 文件夹，找到 Fig Stew。
4. 按住 Control 键点击 Fig Stew 图标，或右键点击 Fig Stew 图标，然后从快捷菜单中选择 Open / 打开。
5. 如果系统提示“无法验证开发者”，再次点击 Open / 打开。

如果以上方法失败，请尝试：

1. 先正常打开一次 Fig Stew，让 macOS 显示阻止提示。
2. 打开 System Settings / 系统设置 -> Privacy & Security / 隐私与安全。
3. 往下找到类似“已阻止使用 Fig Stew，因为无法验证开发者”的提示。
4. 点击 Open Anyway / 仍要打开。

如果仍然提示“已损坏，无法打开”，可以打开 Terminal / 终端，运行：

xattr -dr com.apple.quarantine "/Applications/Fig Stew.app"

然后再重新打开 Fig Stew。

如果仍然无法打开，请邮件联系：figstewissoyummy@proton.me。


3. Windows 如何安装？

1. 双击 Fig Stew Setup 0.1.0-win64.exe。
2. 如果 Windows SmartScreen 提示“未知发布者”，点击 More info / 更多信息，然后点击 Run anyway / 仍要运行。
3. 按照安装向导继续。你可以选择安装位置。
4. 安装完成后运行 Fig Stew。


4. 如何使用？

- 鼠标移到蜜果羹上。
- 按住拖动。
- 鼠标移到四角附近：会出现缩放提示，可以拖拽放大或缩小。
- 右键蜜果羹：打开菜单。
- 右键菜单里的 Garbaniphoro 会打开 https://garbaniphoro.org，这是一个粉丝制作的悬锋城档案馆，收集游戏中与悬锋城相关的文本信息，计划将于2026年6-7月正式发布。
- 右键菜单里的 Say Catchphrase 会让它说话。
- 右键菜单里的 Quit 可以退出。


5. 如何卸载？

Mac：
退出 Fig Stew 后，把 Applications / 应用程序 里的 Fig Stew.app 移到废纸篓；或在 System Settings / 系统设置 -> Storage / 存储 -> Applications / 应用 卸载。

Windows：
打开 Settings / 设置 -> Apps / 应用 -> Installed apps / 已安装的应用，找到 Fig Stew，然后选择 Uninstall / 卸载。


6. 其它

- 由于没有正式签名，macOS 和 Windows 都可能出现安全提示，这并不意味着它存在真实的安全漏洞。感谢您的包容。
- 如果有任何问题，欢迎邮件联系：figstewissoyummy@proton.me


English
--------------------

A Fig Stew desktop pet. It does not do anything except live on your desktop.


1. Which version should I download?

- Apple Silicon Mac (M1 / M2 / M3 / M4, macOS Catalina 10.15 or newer):
  Fig Stew-0.1.0-arm64-adhoc.dmg

- Intel Mac (macOS Catalina 10.15 or newer):
  Fig Stew-0.1.0-x64-adhoc.dmg

- 64-bit Windows PC:
  Fig Stew Setup 0.1.0-win64.exe

If you are not sure which Mac you have:
Click the Apple icon in the top-left corner -> About This Mac.
If the processor says Apple M1/M2/M3/M4, use the Apple Silicon version.
If the processor says Intel, use the Intel Mac version.

If your computer is not on the list above, please email figstewissoyummy@proton.me, and I will send you a suitable version.


2. How to install and open on Mac

Note: This app is not officially signed with an Apple Developer ID yet, so macOS may show a security warning and make the installation process more complicated. This is expected.

1. Open the .dmg file.
2. Drag Fig Stew into the Applications folder.
3. The first time you open it, do NOT double-click it directly. Open the Applications folder in Finder and find Fig Stew.
4. Hold the Control key and click the Fig Stew icon, or right-click the Fig Stew icon, then choose Open from the shortcut menu.
5. If macOS says it cannot verify the developer, click Open again.

If that does not work, try this:

1. Try opening Fig Stew once normally, so macOS shows the blocked-app warning.
2. Open System Settings -> Privacy & Security.
3. Scroll down and find a message similar to "Fig Stew was blocked from use because it is not from an identified developer."
4. Click Open Anyway.

If macOS still says the app is damaged and cannot be opened, open Terminal and run:

xattr -dr com.apple.quarantine "/Applications/Fig Stew.app"

Then open Fig Stew again.

If it still does not work, please email figstewissoyummy@proton.me.


3. How to install on Windows

1. Double-click Fig Stew Setup 0.1.0-win64.exe.
2. If Windows SmartScreen says the publisher is unknown, click More info, then Run anyway.
3. Continue through the installer. You can choose the installation folder.
4. Launch Fig Stew after installation.


4. How to use

- Move your cursor over Fig Stew.
- Click and drag.
- Move your cursor near the corners: resize handles will appear, and you can drag to resize it.
- Right-click Fig Stew: open the menu.
- Garbaniphoro opens https://garbaniphoro.org, a fan-made Castrum Kremnos archive that collects in-game text related to Castrum Kremnos. It is planned to launch officially around June or July 2026.
- Say Catchphrase makes it curse.
- Quit exits the app.


5. How to uninstall

Mac:
Quit Fig Stew, then move Fig Stew.app from Applications to the Trash; or uninstall it from System Settings -> Storage -> Applications.

Windows:
Open Settings -> Apps -> Installed apps, find Fig Stew, and choose Uninstall.


6. Other

- Because the app is not officially signed yet, macOS and Windows may show security warnings. This does not mean it has a real security issue. Thank you for your understanding.
- If you have any questions, please email figstewissoyummy@proton.me
